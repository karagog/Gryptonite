The Cross-Platform Encrypted Password Database

Gryptonite is a very secure and flexible password storage application. It provides an easy to use interface to access and search those passwords when you need them. It runs on any platform because it was developed using Qt.

Features

  • Secure your secrets with a single master password and/or keyfile
  • Conveniently search and access your secrets directly from the tray icon
  • Encrypt and store whole files
  • Generate random passwords easily
  • Bonus utilities include a hash calculator with all major hashes
